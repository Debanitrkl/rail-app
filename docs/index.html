01 / Features

Everything you need, on the move

Designed for the daily railway traveller. From platform-level departures to live maps and home screen glances.

Live Train Tracking

Real-time positions on MapKit maps with interpolated movement between station updates.

MapKit

Journey Management

Flighty-inspired cards with timeline views, delay tracking, and coach position details.

SwiftUI

PNR Status

Instant PNR checks with a watch list for background updates via push notifications.

BullMQ

Station Intelligence

Platform-level departure boards, arrivals, and halt durations for 7,000+ stations.

PostgreSQL

Full-Text Search

Typo-tolerant, instant search across trains, stations, and routes.

Meilisearch

Home Screen Widgets

WidgetKit widgets and Live Activities for at-a-glance status on the lock screen.

WidgetKit

04 / Quick Start

Running in minutes

Clone, configure, compose. Everything is pre-configured for a single-command startup.

Terminal 
# Clone the repository
$ git clone https://github.com/Debanitrkl/rail-app.git

# Configure environment
$ cd rail-app && cp .env.example .env

# Start all services
$ make fresh

# Open the iOS project
$ open Rail/Rail.xcodeproj
