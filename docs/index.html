Rail

The Indian Railway Companion

A full-stack railway tracking application pairing a native iOS client with a robust backend. Real-time train positions, PNR monitoring, station intelligence, and home screen widgets — all wrapped in a premium interface inspired by Flighty.

View Prototype GitHub
Features

Everything you need, on the move

Designed for the daily railway traveller. From platform-level departures to live maps and home screen glances.

Live Train Tracking

Real-time train positions rendered on MapKit maps with interpolated movement between station updates.

MapKit

Journey Management

Flighty-inspired journey cards with timeline views, delay tracking, and coach position details.

SwiftUI

PNR Status

Check PNR status instantly and add PNRs to a watch list for background updates via push notifications.

BullMQ

Station Information

Platform-level departure boards, upcoming arrivals, and halt durations for over 7,000 stations.

PostgreSQL

Full-Text Search

Typo-tolerant, instant search across trains, stations, and routes powered by Meilisearch.

Meilisearch

Home Screen Widgets

WidgetKit widgets and Live Activities for at-a-glance journey status right on the lock screen.

WidgetKit

Quick Start

Up and running in minutes

Clone the repo, start the backend with Docker, and open the Xcode project. Everything is pre-configured.

Terminal 
# Clone the repository
$ git clone https://github.com/Debanitrkl/rail-app.git

# Configure environment
$ cd rail-app && cp .env.example .env

# Start all services (Docker Compose)
$ make fresh

# Open the iOS project
$ open Rail/Rail.xcodeproj
